#  School Pilot

A comprehensive School Management System developed using **PHP** and **MySQL** to automate academic and administrative operations for secondary schools.

---

## Overview

School Pilot is a web-based platform designed to simplify the daily operations of educational institutions by providing centralized management of students, staff, academics, finance, inventory, and school resources.

The system supports role-based access for administrators, teachers, bursars, librarians, nurses, parents, and students, ensuring secure and efficient management of school activities.

## Installation

1. Clone the repository

```bash
git clone https://github.com/robert4techie/school-pilot.git
```

2. Import the database into MySQL.

3. Configure the database connection in:

```
conn.php
```

4. Start Apache and MySQL.

5. Open:

```
http://localhost/school-pilot
```
